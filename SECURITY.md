# Security Policy

## Supported Versions

This project currently supports security fixes on the `main` branch only.

## Reporting a Vulnerability

Please do not open public issues for sensitive vulnerabilities.

Report privately to the maintainer via GitHub: @aurelienlewin.
Include:

- A clear description of the issue.
- Steps to reproduce.
- Potential impact.
- Suggested remediation (if available).

We will acknowledge receipt as soon as possible and coordinate a fix and
responsible disclosure timeline.
